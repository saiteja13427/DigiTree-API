# DigiTree API

> A robust Node backend API for bootcamp directory system

## Usage

Rename "config.env.env" -> "config.env" and put in your values

### Install dependencies

```
npm install
```

### Run App

```
# Run in dev environemnt
npm run dev

#Run in production environment
npm start
```
